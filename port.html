(8Bit Goku) {

[ [Java] [jGrasp] ] {

Java seemed like a foriegn language that I could not understand back in my second year of college. I struggled trying to learn all the classes and methods. But nothing came any eaiser to me than JApplets and GUI's. The artist in me found a moment of relief and success when I discovered a way to use for-loops to create an image, piece by piece. Each for-loop has a counter; when a loop counts to the limit, a small square would appear. Then the next loop would go through the same process to show a different square. Over 4500 lines worth of for-loops perform this action linerally until the whole picture is produced. I presented this as a final project in my Fundemental of Programming Languages 1 course and blew everyone away. After my presentation, I felt as if developing code was do-able.

} 4598 lines of java code to produce the world's greatest Sayin Warrior Goku.
